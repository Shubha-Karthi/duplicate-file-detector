Duplicate File Detector
A Python project that detects duplicate files inside a folder by comparing file contents and hashes. This tool helps users save storage space and organize files efficiently.

Features
- Detects duplicate files automatically
- Uses file hashing for accurate comparison
- Scans folders recursively
- Displays duplicate file paths
- Lightweight and fast
- Beginner-friendly project

Technologies Used
- Python 3
- OS Module
- Hashlib
- File Handling

How It Works
1. The program scans all files in a selected directory.
2. It generates a hash value for each file.
3. Files with matching hashes are identified as duplicates.
4. Duplicate file paths are displayed to the user.
